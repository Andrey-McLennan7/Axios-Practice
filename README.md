# Axios Practice

## Overview
This project is a simple web application built to practice making HTTP requests using **Axios** and rendering API data on the client.  
It demonstrates essential skills for modern web development, including asynchronous JavaScript, API integration, error handling, and dynamic UI updates.

## Features
- Fetches data from an external API using Axios  
- Renders results dynamically in the UI  
- Handles errors gracefully (e.g., failed requests)  
- Clean, simple layout with EJS templates and custom CSS  
- Modular structure that makes it easy to add new endpoints or features

## Tech Stack
- **JavaScript (ES6+)**
- **Axios**
- **EJS**
- **CSS**
- **Node.js**

## Usage
Once the server is running, the app will perform a request to the configured API endpoint and display the returned data on the page.

Typical interactions include:
- Triggering a new Axios request
- Viewing the rendered response
- Seeing error messages if something goes wrong
